# 🏨 Hotel Bookings Dashboard using Power BI

## 📌 Project Overview
This project is an interactive **Hotel Bookings Dashboard** developed using **Power BI** to analyze hotel booking data and generate meaningful business insights for the hospitality industry.

The dashboard helps analyze:
- Hotel booking trends
- Customer behavior
- Revenue performance
- Sales analysis
- Booking status
- User insights
