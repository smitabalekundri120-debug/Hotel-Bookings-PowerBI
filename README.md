# Hotel Bookings Dashboard using Power BI

This project focuses on analyzing hotel booking and transaction data using Power BI to generate interactive dashboards and business insights for the hospitality industry.

## Project Objective
The objective of this project is to analyze hotel bookings, customer transactions, ratings, and hotel performance using data visualization and dashboarding techniques.

## Dataset Files Used
- `bookings.csv`
- `hotels.csv`
- `transactions.csv`
- `users.csv`
- `ratings.csv`

## Tools & Technologies Used
- Power BI
- CSV Datasets
- Data Visualization
- Dashboarding

## Key Analysis Performed
- Hotel booking trend analysis
- Customer transaction analysis
- Hotel ratings analysis
- Revenue and booking insights
- User behavior analysis

## Dashboard Features
- Interactive Power BI dashboards
- KPI cards and visual reports
- Booking and revenue tracking
- Hotel performance comparison
- Customer insights visualization
